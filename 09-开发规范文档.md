**Controller 规范:**

```java
@RestController
@RequestMapping("/sys/user")
@Api(tags = "用户管理")
@Slf4j
public class SysUserController {

    @Autowired
    private SysUserService userService;

    /**
     * 用户列表查询
     *
     * @param queryParam 查询参数
     * @return 用户分页列表
     */
    @GetMapping("/list")
    @ApiOperation(value = "用户列表查询", notes = "分页查询用户信息")
    @ApiImplicitParams({
        @ApiImplicitParam(name = "pageNo", value = "页码", dataType = "int"),
        @ApiImplicitParam(name = "pageSize", value = "每页条数", dataType = "int"),
        @ApiImplicitParam(name = "username", value = "用户名", dataType = "String")
    })
    public Result<Page<SysUser>> list(
            @RequestParam(defaultValue = "1") Integer pageNo,
            @RequestParam(defaultValue = "10") Integer pageSize,
            @RequestParam(required = false) String username) {

        Page<SysUser> page = new Page<>(pageNo, pageSize);
        LambdaQueryWrapper<SysUser> queryWrapper = new LambdaQueryWrapper<>();
        if (StringUtils.isNotBlank(username)) {
            queryWrapper.like(SysUser::getUsername, username);
        }
        queryWrapper.eq(SysUser::getDelFlag, 0);
        queryWrapper.orderByDesc(SysUser::getCreateTime);

        Page<SysUser> result = userService.page(page, queryWrapper);
        return Result.ok(result);
    }

    /**
     * 新增用户
     */
    @PostMapping("/add")
    @RequirePermission("sys:user:add")
    public Result<?> add(@RequestBody @Valid SysUser user) {
        boolean success = userService.addUser(user);
        return success ? Result.ok() : Result.fail("新增失败");
    }

    /**
     * 编辑用户
     */
    @PutMapping("/edit")
    @RequirePermission("sys:user:edit")
    public Result<?> edit(@RequestBody @Valid SysUser user) {
        boolean success = userService.editUser(user);
        return success ? Result.ok() : Result.fail("编辑失败");
    }

    /**
     * 删除用户
     */
    @DeleteMapping("/{id}")
    @RequirePermission("sys:user:delete")
    public Result<?> delete(@PathVariable String id) {
        boolean success = userService.deleteUser(id);
        return success ? Result.ok() : Result.fail("删除失败");
    }
}
```

**Service 规范:**

```java
@Service
@Slf4j
@Transactional(rollbackFor = Exception.class)
public class SysUserServiceImpl implements SysUserService {

    @Autowired
    private SysUserMapper userMapper;

    @Autowired
    private SysUserRoleMapper userRoleMapper;

    @Override
    public boolean addUser(SysUser user) {
        // 1. 参数校验
        if (StringUtils.isBlank(user.getUsername())) {
            throw new BusinessException("用户名不能为空");
        }

        // 2. 检查用户名唯一性
        long count = userMapper.selectCount(
            new LambdaQueryWrapper<SysUser>()
                .eq(SysUser::getUsername, user.getUsername())
                .eq(SysUser::getDelFlag, 0)
        );
        if (count > 0) {
            throw new BusinessException("用户名已存在");
        }

        // 3. 密码加密
        String salt = UUID.randomUUID().toString().replace("-", "");
        user.setSalt(salt);
        user.setPassword(encryptPassword(user.getPassword(), salt));

        // 4. 保存用户
        user.setId(SnowflakeIdWorker.getInstance().nextId());
        user.setCreateTime(new Date());
        user.setDelFlag(0);

        return userMapper.insert(user) > 0;
    }
}
```

### 1.3 SQL 规范

```sql
-- 1. 使用明确的字段列表，不用 SELECT *
SELECT id, username, realname, email, phone
FROM sys_user
WHERE del_flag = 0
ORDER BY create_time DESC
LIMIT 10;

-- 2. 使用 LIMIT 进行分页
SELECT *
FROM sys_user
WHERE del_flag = 0
LIMIT 10 OFFSET 20;

-- 3. 使用 EXPLAIN 分析查询
EXPLAIN SELECT * FROM sys_user WHERE username = 'admin';

-- 4. 创建合适的索引
CREATE INDEX idx_username ON sys_user(username);
CREATE INDEX idx_create_time ON sys_user(create_time);
CREATE INDEX idx_depart_id ON sys_user_depart(depart_id);

-- 5. 批量插入
INSERT INTO sys_user (id, username, password, create_time) VALUES
('1', 'user1', 'pwd1', NOW()),
('2', 'user2', 'pwd2', NOW()),
('3', 'user3', 'pwd3', NOW());

-- 6. 使用事务
START TRANSACTION;

UPDATE sys_user SET status = 2 WHERE id = '1';
UPDATE sys_user SET update_time = NOW() WHERE id = '1';

COMMIT;

```

```
       │               │               │
       ▼               ▼               ▼
```

**分支命名规范:**

| 分支类型 | 命名格式 | 示例 |
|----------|----------|------|
| 主分支 | main | main |
| 开发分支 | develop | develop |
| 预发布分支 | release/x.x.x | release/1.0.0 |
| 功能分支 | feature/功能名 | feature/user-management |
| Bug修复分支 | fix/问题描述 | fix/login-bug |
| 热修复分支 | hotfix/问题描述 | hotfix/critical-bug |

### 2.2 Commit 规范

**提交信息格式:**

```
<type>(<scope>): <subject>

<body>

<footer>
```

**Type 类型:**

| 类型 | 说明 |
|------|------|
| feat | 新功能 |
| fix | Bug修复 |
| docs | 文档更新 |
| style | 代码格式（不影响功能） |

**提交示例:**

```bash

```

### 2.3 Pull Request 流程

```bash
# 1. 从 develop 创建功能分支
git checkout develop
git pull origin develop
git checkout -b feature/user-management

# 2. 开发并提交
git add .
git commit -m "feat(user): 新增用户 CRUD 接口"

# 3. 推送分支
git push origin feature/user-management

# 4. 创建 Pull Request
# 在 GitLab/GitHub 界面创建 PR
# 选择源分支: feature/user-management
# 选择目标分支: develop
# 填写 PR 描述

# 5. Code Review 后合并
# 使用 Squash and Merge 保持提交历史整洁
```

### 3.1 RESTful API 设计

**URL 命名规范:**

| 操作 | 方法 | URL | 说明 |
|------|------|-----|------|
| 列表 | GET | /sys/users | 用户列表 |
| 详情 | GET | /sys/users/{id} | 用户详情 |
| 新增 | POST | /sys/users | 新增用户 |
| 编辑 | PUT | /sys/users/{id} | 编辑用户 |
| 删除 | DELETE | /sys/users/{id} | 删除用户 |

**URL 设计原则:**

```
# 资源命名使用复数
✅ /sys/users
❌ /sys/user

# 使用名词, 不使用动词
✅ /sys/users/{id}
❌ /sys/getUser

# 使用连字符分隔
✅ /open-services
❌ /openServices

# 参数使用小写
✅ /sys/users?status=active
❌ /sys/users?Status=Active

# 层级关系
✅ /sys/users/{userId}/roles  # 获取用户的角色
✅ /sys/departments/{deptId}/users  # 获取部门下的用户
```

## 四、异常处理规范

### 4.1 异常分类

```java
/**
 * 业务异常
 */
public class BusinessException extends RuntimeException {
    private int code = 3001;

    public BusinessException(String message) {
        super(message);
    }

    public BusinessException(int code, String message) {
        super(message);
        this.code = code;
    }
}

/**
 * 未登录异常
 */
public class UnauthorizedException extends RuntimeException {
    public UnauthorizedException() {
        super("未登录或登录已过期");
    }
}

/**
 * 无权限异常
 */
public class NoPermissionException extends RuntimeException {
    public NoPermissionException() {
```

### 4.2 全局异常处理

```java
@RestControllerAdvice
@Slf4j
public class GlobalExceptionHandler {

    /**
     * 业务异常处理
     */
    @ExceptionHandler(BusinessException.class)
    public Result<?> handleBusinessException(BusinessException e) {
        log.warn("业务异常: {}", e.getMessage());
        return Result.fail(e.getCode(), e.getMessage());
    }

    /**
     * 认证异常处理
     */
    @ExceptionHandler(UnauthorizedException.class)
    public Result<?> handleUnauthorizedException(UnauthorizedException e) {
        log.warn("认证异常: {}", e.getMessage());
        return Result.fail(2001, e.getMessage());
    }

    /**
     * 权限异常处理
     */
    @ExceptionHandler(NoPermissionException.class)
    public Result<?> handleNoPermissionException(NoPermissionException e) {
        log.warn("权限异常: {}", e.getMessage());
        return Result.fail(2003, e.getMessage());
    }

    /**
     * 参数校验异常处理
     */
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<?> handleValidationException(MethodArgumentNotValidException e) {
        String message = e.getBindingResult().getFieldError().getDefaultMessage();
        log.warn("参数校验异常: {}", message);
        return Result.fail(1001, message);
    }

    /**
     * SQL 异常处理
     */
    @ExceptionHandler(DataIntegrityViolationException.class)
    public Result<?> handleDataIntegrityViolationException(DataIntegrityViolationException e) {
        log.error("数据完整性异常: {}", e.getMessage());
        if (e.getMessage().contains("Duplicate entry")) {
            return Result.fail(3003, "数据已存在");
        }
        return Result.fail("数据操作失败");
    }

    /**
     * 其他异常处理
     */
    @ExceptionHandler(Exception.class)
    public Result<?> handleException(Exception e) {
        log.error("系统异常: ", e);
        return Result.fail(5000, "系统内部错误");
    }
}
```
