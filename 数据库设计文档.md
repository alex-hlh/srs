# 数据库设计文档

## 1. 数据库概述

- **数据库类型**: MySQL 8.0+ / 达梦 / Oracle / PostgreSQL 等多数据库支持
- **主键策略**: 雪花算法 (ASSIGN_ID)
- **逻辑删除**: 统一使用 del_flag 字段

## 4. 优化清单

### 4.1 删除的冗余字段

| 表名 | 字段名 | 删除原因 |
|------|--------|----------|
| sys_user | org_code_txt | transient字段,仅用于前端显示 |
| sys_user | activiti_sync | 预留字段,工作流已不维护 |
| sys_user | client_id | 设备ID,推送功能已不维护 |

### 4.2 字段重命名

| 原字段名 | 新字段名 | 原因 |
|----------|----------|------|
| sys_announcement.titile | title | 拼写错误 |

### 4.3 统一规范

1. **时间字段**: 所有表统一使用 `create_time` 和 `update_time`
2. **创建人字段**: 所有表统一使用 `create_by` (varchar类型)
3. **逻辑删除**: 所有表统一使用 `del_flag` (tinyint, 0未删 1已删)
4. **状态字段**: 统一使用 `status` (int, 1启用 0禁用)
