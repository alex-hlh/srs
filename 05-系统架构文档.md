### 2.3 请求处理流程

```
请求流程:

浏览器
    │
    │ https://api.example.com/ctrl/system/user/list
    │
    ▼
Nginx 负载均衡
    │
    │ 代理到 srs-gateway:8080
    ▼
Spring Cloud Gateway
    │
    ├── AuthFilter (验证 Token)
    ├── RateLimitFilter (限流)
    ├── LogFilter (记录日志)
    │
    ▼
路由转发 (根据路径 /ctrl/*)
    │
    │ proxy_pass http://srs-ctrl:8082
    ▼
FtCtrlApplication
    │
    ├── DispatcherServlet
    ├── HandlerMapping (映射 Controller)
    ├── HandlerInterceptor (拦截器链)
    │   ├── LogInterceptor (日志)
    │   └── AuthInterceptor (权限)
    │
    ▼
Controller (SysUserController)
    │
    ▼
Service (SysUserService)
    │
    ▼
MyBatis-Plus Mapper
    │
    ▼
MySQL Database
```

### 3.3 权限控制流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      权限控制流程                                  │
└─────────────────────────────────────────────────────────────────┘

1. 登录获取 Token
   ┌─────────┐    POST /sys/login    ┌─────────┐
   │  登录页  │ ────────────────────▶ │  后端   │
   └─────────┘                       └────┬────┘
       │                                   │
       │  ◀────────────── Token + 菜单树 ──┘
       │
       ▼
2. 存储用户信息
   ┌─────────┐
   │ Vuex    │ ─── store.commit('SET_TOKEN', token)
   │ user    │ ─── store.commit('SET_USER_INFO', info)
   │         │ ─── store.commit('SET_PERMISSIONS', permissions)
   └─────────┘
       │
       ▼
3. 生成动态路由
   ┌─────────────┐
   │ router      │ ─── store.getters.permission_routes
   │ (Vue Router)│ ─── generateRoutes(permissions)
   └─────────────┘
       │
       ▼
4. 路由守卫控制
   ┌─────────────┐
   │ Navigation  │ ─── beforeEach((to, from, next) => {
   │   Guard     │      if (!token && to.path !== '/login') {
   │             │        next('/login')
   │             │      }
   │             │    }
   │             │   })
   └─────────────┘
       │
       ▼
5. 页面级权限指令
   ┌─────────────┐
   │ v-permission│ ─── v-if="hasPermission('sys:user:add')"
   │ directive   │
   └─────────────┘
```
