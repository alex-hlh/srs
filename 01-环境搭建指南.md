## 二、后端环境搭建

### 2.1 JDK 安装配置

```bash
# macOS 使用 Homebrew
brew install openjdk@17

# Linux (RPM)
sudo rpm -ivh jdk-17_linux-x64_bin.rpm

# Windows 下载安装包直接安装
```

**环境变量配置 (Linux/macOS):**

```bash
# ~/.bash_profile 或 ~/.zshrc
export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH
export MAVEN_HOME=/opt/maven
export PATH=$MAVEN_HOME/bin:$PATH
```

**验证安装:**

```bash
java -version
# openjdk version "17.0.x" ...

mvn -version
# Apache Maven 3.8.x ...
```

### 2.2 Maven 配置

**settings.xml 配置 (~/maven/conf/settings.xml):**

```xml
<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0
                              http://maven.apache.org/xsd/settings-1.0.0.xsd">

    <mirrors>
        <!-- 内部私服镜像 -->
        <mirror>
            <id>srs-maven</id>
            <name>SRS Private Repository</name>
            <url>http://xxx.xxx.xxx.xxx:8081/repository/maven-public/</url>
            <mirrorOf>central</mirrorOf>
        </mirror>
    </mirrors>

    <profiles>
        <profile>
            <id>jdk-17</id>
            <activation>
                <activeByDefault>true</activeByDefault>
                <jdk>17</jdk>
            </activation>
            <properties>
                <maven.compiler.source>17</maven.compiler.source>
                <maven.compiler.target>17</maven.compiler.target>
                <maven.compiler.compilerVersion>17</maven.compiler.compilerVersion>
            </properties>
        </profile>
    </profiles>
</settings>
```

### 2.3 数据库初始化

```bash
# 1. 创建数据库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS srs_db DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 2. 导入初始化脚本
mysql -u root -p xx_ft < /path/to/init_mysql.sql

# 3. 验证导入
mysql -u root -p -e "USE xx_ft; SHOW TABLES;"
```

### 2.4 中间件启动

**Redis:**

```bash
# macOS
brew services start redis

# Linux
sudo systemctl start redis

# 验证
redis-cli ping
# PONG
```

**RocketMQ 单机部署:**

```bash
# 1. 下载 RocketMQ 4.9.x
cd /opt
wget https://archive.apache.org/dist/rocketmq/4.9.4/rocketmq-all-4.9.4-bin-release.zip
unzip rocketmq-all-4.9.4-bin-release.zip

# 2. 启动 NameServer
cd rocketmq-all-4.9.4-bin-release
nohup sh bin/mqnamesrv &

# 3. 启动 Broker
nohup sh bin/mqbroker -n localhost:9876 &

# 验证
tail -f logs/rocketmqlogs/namesrv.log
```

**MinIO 单机部署:**

```bash
# 1. 下载
wget https://dl.min.io/server/minio/release/linux-amd64/minio
chmod +x minio

# 2. 启动
MINIO_ROOT_USER=minioadmin MINIO_ROOT_PASSWORD=minioadmin ./minio server /data --console-address ":9001"

# 3. 控制台访问
# http://localhost:9001
# 用户名: minioadmin
# 密码: minioadmin
```

### 2.5 后端项目导入

**IDEA 导入步骤:**

1. File → Open → 选择 `srs-platform` 目录
2. 选择 Import as Maven project
3. 等待 Maven 索引下载完成（约 10-15 分钟）
4. 配置 SDK: File → Project Structure → Project → SDK → 选择 JDK 17
5. 配置 Maven: File → Settings → Build → Maven → 选择 Maven 主目录和 settings.xml

**Maven 命令行编译:**

```bash
cd srs-platform

# 编译整个项目
mvn clean install -DskipTests

# 编译指定模块
mvn -pl srs-center -am clean compile

# 运行主模块
cd srs-ctrl
mvn spring-boot:run
```

## 三、前端环境搭建

### 3.1 Node.js 安装

```bash
# macOS 使用 nvm 管理多版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
nvm alias default 18

# Windows 使用 nvm-windows
# 下载: https://github.com/coreybutler/nvm-windows/releases
```

### 3.2 Yarn 安装

```bash
# npm 全局安装
npm install -g yarn

# 验证
yarn --version
# 1.22.x
```

### 3.3 项目依赖安装

```bash
cd srs-ctrl-ui

# 安装依赖 (约 5-10 分钟)
yarn install

# 如果安装慢,配置淘宝镜像
yarn config set registry https://registry.npmmirror.com
yarn install
```

### 3.4 前端开发服务器

```bash
# 开发模式启动
yarn serve

# 访问地址
# http://localhost:3000

# 构建生产版本
yarn build

# 构建测试版本
yarn build:test
```

## 四、关键配置修改

### 4.1 后端配置文件

**ft-ctrl/src/main/resources/application.yml:**

```yaml
server:
  port: 20000

spring:
  application:
    name: ft-ctrl

  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/srs_db?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password

  redis:
    host: localhost
    port: 6379
    password:
    database: 0

 rocketmq:
    namesrv-addr: localhost:9876
    producer:
      group: ft-producer-group

  elasticsearch:
    uris: http://localhost:9200

  minio:
    endpoint: http://localhost:9000
    access-key: minioadmin
    secret-key: minioadmin
    bucket-name: xx-ft
```

**ft-ctrl-ui/.env.development:**

## 六、常见问题

### 6.1 Maven 下载慢

```bash
# 配置国内镜像
mvn clean install -DskipTests -Dmaven.repo.local=/path/to/local/repo
```

### 6.2 Node Sass 安装失败

```bash
# 使用淘宝镜像
yarn config set sass_binary_site https://npm.taobao.org/mirrors/node-sass
yarn install
```

### 6.3 端口被占用

```bash
# macOS
lsof -i :3000
kill -9 <PID>

# Linux
netstat -tlnp | grep 3000
```

### 6.4 MySQL 连接失败

```bash
# 检查 MySQL 服务
mysql -u root -p -e "SELECT 1"

# 检查用户权限
mysql -u root -p -e "GRANT ALL PRIVILEGES ON srs_db.* TO 'root'@'localhost';"
```
