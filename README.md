## 项目复刻清单

### 环境准备 (1-2天)
- [ ] 安装 JDK 17
- [ ] 安装 Maven 3.8+
- [ ] 安装 Node.js 18
- [ ] 安装 Docker Desktop
- [ ] 克隆代码仓库

### 中间件搭建 (1天)
- [ ] 启动 MySQL 8.0
- [ ] 启动 Redis 7.x
- [ ] 启动 Nacos 2.x
- [ ] 启动 RocketMQ 4.9.x
- [ ] 启动 Elasticsearch 7.17.x
- [ ] 启动 MinIO

### 数据库初始化 (0.5天)
- [ ] 创建数据库 `xx_ft`
- [ ] 执行 init_mysql.sql
- [ ] 验证表创建成功

### 后端启动 (0.5天)
- [ ] 导入 ft-platform 到 IDEA
- [ ] 修改 application-dev.yml 配置
- [ ] 编译项目 `mvn clean install`
- [ ] 启动 ft-center
- [ ] 启动 ft-ctrl
- [ ] 启动 ft-gateway
- [ ] 验证接口可用

### 前端启动 (0.5天)
- [ ] 进入 ft-ctrl-ui 目录
- [ ] 执行 `yarn install`
- [ ] 修改 .env.development 配置
- [ ] 启动 `yarn serve`
- [ ] 访问 http://localhost:3000
- [ ] 验证登录功能

### 功能验证 (1-2天)
- [ ] 管理员登录
- [ ] 用户管理 CRUD

- [ ] 权限管理

## 问题排查

| 问题 | 解决方案 |
|------|----------|
| Maven 下载依赖慢 | 配置国内镜像 |
| Node Sass 安装失败 | 配置淘宝镜像 |
| 端口被占用 | `lsof -i:端口号` 查找并 kill |
| MySQL 连接失败 | 检查用户名密码和权限 |
| Redis 连接失败 | 检查防火墙和密码配置 |
| 前端 API 请求失败 | 检查代理配置和后端服务 |
