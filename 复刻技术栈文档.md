> 本文档为产品经理和开发团队提供SRS项目完整技术栈参考，用于项目复刻或类似系统开发。

## 一、项目概述

星环空间 (Stellar Ring Space, SRS) 是**SRS软件**的企业级SaaS办公系统，采用**前后端分离架构**：

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Vue 2.6 + Ant Design Vue | 管理后台Web界面 |
| 后端 | Spring Boot 2.5.4 + Java 8 | RESTful API服务 |
| 数据库 | 多数据库支持（达梦/MySQL/Oracle/PostgreSQL等） | 国产化适配 |
| 基础设施 | Redis + RocketMQ + Elasticsearch + MinIO | 缓存/消息/搜索/存储 |

## 四、项目模块结构

```
srs-platform/
├── common/                    # 公共模块
│   ├── base/                  # 基础框架包（集成Web/Redis/Quartz等）
│   ├── component/
│   │   ├── redis/            # Redis组件
│   │   ├── fileStorage/       # MinIO文件存储组件
│   │   ├── elasticSearch/     # ES搜索组件
│   │   └── rocketMQ/          # MQ消息组件
│   └── utils/                 # 工具类
├── srs-center/                 # 统一认证中心（登录/权限/SSO）
├── srs-ctrl/                   # 智能管控平台（业务功能）
├── srs-gateway/                 # 安全网关
├── ft-doc-center/             # 文档中心
└── ft-meeting/                # 视频会议
```

## 六、开发与构建命令

### 前端

```bash
cd srs-ctrl-ui
yarn install              # 安装依赖
yarn serve               # 开发模式 (localhost:3000)
yarn build               # 生产构建
yarn build:test          # 测试环境构建
yarn lint                # 代码检查
```

### 后端

```bash
cd srs-platform
mvn clean install        # 编译整个项目
mvn -pl srs-center -am compile   # 编译指定模块
mvn test                 # 运行测试
```
